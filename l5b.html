- Lesson 5 -
SQL - Relationship queries (JOIN)

Summary of different types of JOIN and their syntax

  • FULL OUTER JOIN (Inclusive)

    Select all records in tables A and B

    FULL OUTER JOIN (Exclusive)

    Select all records in tables A and B, except the ones existing in both

  • LEFT JOIN (Inclusive)

    Select all records in table A, even if they do not exist in table B

    RIGHT JOIN (Inclusive)

    Select all records in table B, even if they do not exist in table A

  • LEFT JOIN (Exclusive)

    Select the records existing only in table A but not in table B

    RIGHT JOIN (Exclusive)

    Select the records existing only in table B but not in table A

  • INNER JOIN

    Select the records existing in both table A and table B

    Self Join

    Join a table to itself by making two copies of it
